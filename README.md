# Bank Customer Churn Prediction System

## 📌 Project Overview
This project predicts customer churn in the banking sector using machine learning and provides decision support for retention strategies.

## 🚀 Features
- Churn probability prediction
- Risk segmentation (High / Medium / Low)
- Feature importance analysis
- Value at Risk (VaR) calculation
- Decision support system
- Streamlit dashboard

## 🛠 Technologies Used
- Python
- Scikit-learn
- XGBoost
- Pandas, NumPy
- Matplotlib, Seaborn
- Streamlit

## 📊 Model Performance
- Accuracy: 97%
- Recall: 90%
- AUC: 0.99

## 💻 How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run model:
   python main.py

3. Run app:
   streamlit run app.py
